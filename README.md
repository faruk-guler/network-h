# Network for Humanity by "faruk-guler"

## Network (Ağ) Nedir?

**Network (Ağ)**, iki veya daha fazla cihazın **veri paylaşmak** için birbirine bağlandığı yapıdır. İnternet, dünyanın en büyük ağıdır!

---

| Sistem | Slogan | Kök | Örnek |
|---|---|---|---|
| Binary | Elektrik var ya da yok | Latince bīnārius (İkişerli) | `11000000.10101000.00000001.00000001` |
| Decimal | İnsan algısının ölçeği | Latince *Decem* (On) | `192.168.1.1` |
| Hexadecimal | Sistemin melez köprüsü | *Hex* (6) + *Decem* (10) a, b, c .. f [toplam 16 karakter | `00:1A:2B:3C:4D:5E` (MAC) |
| Octect | yunanca 8 parcalı demektir| | her octect min 0 max en fazla 255 değerini alabilir |
