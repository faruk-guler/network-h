## Network (Ağ) Nedir?

**Network (Ağ)**, iki veya daha fazla cihazın **veri paylaşmak** için birbirine bağlandığı yapıdır. İnternet, dünyanın en büyük ağıdır!

 ```bash
root@guler-host:~# network -h
- images
- ip-address
- models
- .....
root@guler-host:~#
 ```
