# network for humanity by "faruk-guler"

### BINARY (İkili Sistem)

| Özellik | Açıklama |
|--------|----------|
| Tema | Makine Dilinin Saf Hali |
| Kök | Latince *Bini* (İkişerli) |
| Felsefe | Dyad (1 ve 0). Varlık ve yokluk dengesi. |
| Mantık | Elektrik Var (1) / Elektrik Yok (0) |
| Örnek | 11000000.10101000.00000001.00000001 |

### DECIMAL (Onluk Sistem)

| Özellik | Açıklama |
|--------|----------|
| Tema | İnsan Algısının Ölçeği |
| Kök | Latince *Decem* (On) |
| Felsefe | İnsan odaklı düzen. On parmağın yansıması. |
| Mantık | Veriyi anlamlandırmak için tercüme arayüzü. |
| Örnek | 192.168.1.1 |

### HEXADECIMAL (Onaltılık Sistem)

| Özellik | Açıklama |
|--------|----------|
| Tema | Sistemin Melez Köprüsü |
| Kök | Yunanca *Hex* (6) + Latince *Decem* (10) |
| Felsefe | Estetik sıkıştırma. Karmaşıklığı sembolleştirme. |
| Mantık | Binary dizilerini yönetilebilir forma sokma. |
| Örnek | fe80::1 (IPv6), 00:1A:2B:3C:4D:5E (MAC) |
